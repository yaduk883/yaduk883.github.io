LEARN AN INDIAN LANGUAGE (LAIL)

Choose your native language, pick one or more target languages to learn, explore letters and step-by-step basic sentences with transliteration. (You can add more languages later.)

Languages included (you can add more later): Malayalam, Tamil, Kannada, Telugu, Hindi, Odia, Punjabi, Sanskrit, Bodo, Assamese, English.

Letters / Alphabet

Click a language from the selected list (below) to view its letters. These are starter letters — you can edit the lists in the code comments.

Basic Sentences (Step by Step)

Each step shows a set of sentences. Click Next to go to harder sentences. You can add transliteration for each sentence in the input field below the sentence.

Step: 1

Notes: Each sentence card contains: target-language sentence, English translation, input for transliteration, and a checkbox to mark "mastered".

Compare Languages

Select multiple languages to compare the same sentence across them (side-by-side).

Settings & Help

You can edit the initial letters and sentence lists inside the LANG_DATA object in the script. Comments in the code explain where to add new languages and sentences.

All data is stored in browser memory (localStorage) so your edits persist on this device. To reset, clear browser storage for this site.
