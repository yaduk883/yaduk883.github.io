✨ What is LAIL?

LAIL (Learn an Indian Language) is a tool to help you learn multiple Indian languages side‑by‑side. You can:

  • Compare languages
  • Learn alphabets
  • Practice sentences step‑by‑step
  • Save transliterations
  • Mark learned sentences
